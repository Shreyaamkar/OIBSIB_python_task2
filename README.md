# OIBSIB_python_task2
BMI CALCULATOR

✅ Code 1: Flask Backend (BMI Calculation API)
Objective: To calculate Body Mass Index (BMI) and determine health category based on user input.
Tools: Python, Flask, JSON handling.
Outcome: A backend API (/calculate) that returns BMI value and weight category in JSON format.

✅ Code 2: HTML + JavaScript Frontend
Objective: To collect user height and weight and display BMI results interactively.
Tools: HTML5, JavaScript, Fetch API.
Outcome: A simple web interface that sends data to the server and displays BMI results instantly.

✅ Code 3: CSS Styling
Objective: To enhance the visual appearance and usability of the BMI calculator interface.
Tools: CSS3.
Outcome: A clean, centered, and user-friendly BMI calculator design.
